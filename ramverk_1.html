Om Bootstrap

Bootstrap har en "mobile-first" kärna; vilket innebär att stylingen är uppbyggd med utgångspunkt för mobilvisning och sedan anpassas när webbsidan förstoras, istället för tvärtom.
För att använda Bootstrap behöver man först inkludera filerna, i detta projekt har jag valt att göra detta med hjälp av en CDN, vilket gör att jag inte först behöver hämta hem filerna lokalt för att kunna använda dem.
För att styla sin webbsida med bootsstrap så använder man olika klasser för att välja vilken typ av layout man önskar på sina element.
Innehållet delas även in i mindre grupper med hjälp av div-element för att få ett särskilt utseende.
Bootstrap var ganska enkelt att använda, och klasserna som används var enligt min åsikt relativt självförklarande.
