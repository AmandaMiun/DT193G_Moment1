GitHub Pages

För att publicera min webbplats har jag använt mig av GitHub Pages. Detta gör det möjligt att skapa en statisk webbplats direkt i GitHub.
Anledningen till att jag valde denna metod var att vi tidigare under kursen har använt GitHub för versionshantering, och jag ville då testa denna metod för publicering också.
GitHub pages är gratis att använda, vilket är en stor fördel gentemot andra alternativ.
Det stödjer programmeringsspråken HTML, CSS och JavaScript. Som jag tolkade det på deras hemsida kan man inte använda sig av exempelvis PHP, på grund av ett tjänsten endast tillåter publicering av statiska webbplatser.

För att publicera en sida på GitHub så skapar man ett repo precis som vanligt, sedan går man in på sitt repo i GitHub och navigerar fram till "pages" under inställningar.
Där väljer man sedan vilken "branch" man vill publicera och trycker på "save".
För att se sin publicerade webbplats navigerar men till adressen: användarnamn.github.io/repositorynamn/
