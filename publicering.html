GuitHub Pages

För att publicera min webbplats har jag använt mig av GitHub Pages. Detta gör det möjligt att skapa en statisk webbplats direkt i GitHu.
Anledningen till att jag valde denna metod var att vi tidigare under kursen har använt GitHub för versionehantering, och jag ville då testa denna metod för publicering också.
GitHub pages är gratis att använda, vilket är en stor fördel gentemot andra alternativ.
Det stödjer programmeringsspråken HTML, CSS och JavaScript. Som jag tolkade det på deras hemsida kan man inte använda sig av exempelvis PHP, på grund av ett tjänsten endast tillåter publicering av statiska webbplatser.
